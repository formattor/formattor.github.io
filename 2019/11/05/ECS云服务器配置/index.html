ECS云服务器配置
发表于 2019-11-05| 更新于 2021-03-30|配置
字数总计: 396|阅读时长: 1 分钟|阅读量:

由于python2.7升级python3产生故障，导致yum源不可用，虽然配置好了但是又导致了宝塔不可用，由于本服务器是拿来练手的，故重置了一下，由于用的不多，故要配置的也不多，记录一下。

vim配置

永久显示代码行

  • cd /etc
  • vim vimrc
  • 在最后一行填写
  • 按ESC : wq 即可

tab键设置为4

  • vim /etc/vimrc
  • 在最后一行填写
  • 按ESC : wq 即可

linux配置

主机名修改

  • hostnamectl set-hostname 主机名
  • 重启实例

初始化实例

  • 登陆ECS控制台
  • 左侧导航栏，点击实例与镜像>实例
  • 找到要初始化的实例
  • 点击ID进入实例详情
  • 点击停止
  • 左侧导航栏，单击本实例磁盘
  • 点击右侧初始化磁盘
  • 输入进入实例时的密码
  • 确认即可
    参考连接

配置宝塔

  • 安装(确保系统纯净，未安装lamp或者lnmp)
    文档地址
  • 进入实例，输入bt default
  • 将url在浏览器打开，就会出现宝塔面板
  • 如果未跳转成功，则是云服务器的8888端口没有向外开放，在实例的安全组规则下配置即可
    • 添加安全组规则
    • 填入端口范围为8888
    • 授权对象为0.0.0.0/0
    • 确认即可
  • 第一次进入输入bt default下给的用户名和密码
  • 安装lamp或者lnmp(确保未安装)
