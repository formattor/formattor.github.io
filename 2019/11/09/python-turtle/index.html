python-turtle
发表于 2019-11-09| 更新于 2019-11-09
字数总计: 179|阅读时长: 1 分钟|阅读量:

配置

  • 环境：python3.7
  • 编译软件：subline text
  • 需要的包：turtle

用法

  • 引入
  • 函数
    • pendown() #落笔
    • penup() #提笔
    • forword(200) #向前移动200
    • rigth(90) #右转90°
    • speed() #参数1~10速度递增，0速度最快
    • reset() #重置配置
    • circle() #圆，负参数为顺时针，正参数为逆时针
      • circle(100,180) #逆时针，半径100，半圆
      • circle(100,360,4) #第三个参数为，棱角数
    • fillcolor(params color) #首先设置颜色参数
      • begin_fill() #填充颜色
      • end_fill() #结束填充
    • pencolor(params color) #画笔颜色
    • goto(x,y) #前往位置
    • undo() #撤回一步
    • dot() #点，参数为直径
    • clear() #清理画布
    • pos() #获取画笔位置
    • setpos() #设置画笔位置
    • heading() #获取角度
    • seth() #设置角度
