笔记

新建笔记时遇到了很多bug,不了解为何上传到github的仅有public部分，那hexo new到source/_posts目录还有什么意义。
还是没有搞清hexo的工作原理，hexo generate的命令就是在public生成静态页面的，到日期目录下吧.md转换成.html。
这个bug浪费了我一下午时间，我觉得是我的学习方式问题，看问题仅仅浮于表面，sigh!!!
再次记录一下吧

  • hexo new “postName” #新建文章
  • hexo new page “pageName” #新建页面
  • hexo generate #生成静态页面至public目录```
  • hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
  • hexo deploy #部署到GitHub
  • hexo help # 查看帮助
  • hexo version #查看Hexo的版本
